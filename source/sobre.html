Sobre o projeto

O ForumCloud é uma aplicação de fórum online desenvolvida como parte do Trabalho 03 da disciplina de Cloud Computing da Unidavi, simulando a infraestrutura cloud de uma plataforma de discussão colaborativa.

O ambiente é executado em um container Docker com Ubuntu Server e Apache, com deploy automatizado via Shell Script, seguindo práticas reais de DevOps e administração de sistemas Linux.
